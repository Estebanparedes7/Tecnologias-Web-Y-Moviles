# Actividad 2 Flutter

Esta es una aplicación Flutter que muestra información sobre los primeros 3 pokemon del pokedex
## Cómo funciona

La aplicación consta de tarjetas que representan diferentes pokemon con su propia imagen, nombre y tipo.
